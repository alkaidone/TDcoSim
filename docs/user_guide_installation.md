# Installation
From github::

    git clone https://github.com/deephyper/deephyper.git
    cd deephyper/
    pip install -e .

### Dependencies:
* Python, version: 2.7.5

* Python packages: SciPy, Numpy, Matlplotlib, XlsxWriter
