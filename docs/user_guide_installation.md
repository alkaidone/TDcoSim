# Installation
From GitHub::

    git clone https://github.com/tdcosim/TDcoSim.git
    cd tdcosim/
    pip install -e .

### Dependencies:
* [Python, version = 2.7.5](https://www.python.org/)

* Python packages: SciPy, Numpy, Matlplotlib, Pywin32, XlsxWriter
